## Installation

```bash
$ npm install
```

## Environments
> Copy ".env-example" and rename to ".env".
> Enter your environments.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Docker

```bash
# docker build
$ npm run docker:build

# docker run
$ npm run docker:run

# docker compose run
$ npm run docker:compose
```
